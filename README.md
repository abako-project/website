# Polkatalent

Projecto Work3Spaces

## Lanzar el backend

Probado en los siguientes sistemas operativos:
* macOS Sequoia 15.5:
* macOS Tahoe 16.0.1:

Instalaciones requeridas:
* node 22, 24

Para lanzar el backend, ejecutar los siguientres comandos:
```
cd backend
npm install
npm run migrate
npm run seed
npm start
```

Y lanzar un navegador web y visitar la URL:

```
http://localhost:3001
```
